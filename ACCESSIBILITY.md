# Режим для слабовидящих

## Описание

Приложение поддерживает специальный режим для слабовидящих пользователей с увеличенными шрифтами и высококонтрастной цветовой схемой.

## Возможности

- **Увеличенные шрифты**: все текстовые элементы увеличиваются на 30% (коэффициент 1.3)
- **Высокий контраст**: черный текст на белом фоне с контрастными акцентами
- **Увеличенные элементы управления**: кнопки и поля ввода автоматически увеличиваются
- **Сохранение настроек**: выбор режима сохраняется между сеансами

## Включение режима

1. Откройте экран "Профиль" в нижней навигации
2. Найдите переключатель "Версия для слабовидящих"
3. Включите переключатель
4. Приложение автоматически применит настройки

## Технические детали

### Компоненты

- `AccessibilityProvider` - контекст для управления состоянием режима
- `useAccessibility()` - хук для доступа к режиму
- `useAccessibilityFontSize(baseSize)` - хук для масштабирования шрифтов
- `useThemeColors()` - хук для получения цветов с учетом режима

### Цветовая схема

```typescript
Colors.highContrast = {
  text: '#000000',
  background: '#FFFFFF',
  primary: '#0066CC',
  border: '#000000',
  // ...
}
```

### Масштабирование

```typescript
AccessibilityFontScale = 1.3
```

## Поддерживаемые компоненты

Все основные UI компоненты автоматически поддерживают режим для слабовидящих:

- `ThemedText` - автоматическое масштабирование шрифтов
- `ThemedView` - высококонтрастные цвета
- `Button` - увеличенная высота и шрифт
- `Input` - увеличенная высота и шрифт
- `Card` - высококонтрастные границы

## Экраны

Все экраны приложения поддерживают режим для слабовидящих:

- Главная
- Пациенты
- Уведомления
- Профиль
- Вход
- Регистрация
