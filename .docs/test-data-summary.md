## 🔧 Дополнительные возможности

### Загрузка документов
Документы можно загрузить вручную через приложение:
1. Открыть карточку пациента
2. Перейти на вкладку "Документы"
3. Нажать кнопку "Загрузить"
4. Выбрать файл (фото, PDF, документ)
5. Указать категорию (анализы, снимки, другое)

### Изменение статусов
Демонстрация workflow:
- **NEW** → **PREPARATION** → **REVIEW_NEEDED** → **APPROVED** → **SURGERY_SCHEDULED** → **COMPLETED**

Текущие пациенты находятся в статусе **PREPARATION**, можно перевести их в **REVIEW_NEEDED** для демонстрации проверки хирургом.

### Создание новых пациентов
1. Нажать кнопку "+" на вкладке "Пациенты"
2. Заполнить форму
3. Система автоматически создаст код доступа для пациента
